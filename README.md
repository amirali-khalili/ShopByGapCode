# فروشگاه پوشاک آنلاین - Django MVT

یک فروشگاه پوشاک کامل با معماری Django MVT، طراحی RTL فارسی و رابط کاربری مدرن.

## ویژگی‌ها

### عملکرد اصلی
- ✅ معماری استاندارد Django MVT
- ✅ قالب RTL فارسی با فونت Vazirmatn
- ✅ طراحی مدرن و زیبا با Bootstrap 5 RTL
- ✅ کاملاً ریسپانسیو (موبایل، تبلت، دسکتاپ)

### صفحات
- 🏠 **صفحه اصلی**: نمایش محصولات ویژه، دسته‌بندی‌ها و ویژگی‌های فروشگاه
- 📦 **لیست محصولات**: نمایش تمام محصولات با فیلتر و جستجو
- 🔍 **جزئیات محصول**: اطلاعات کامل محصول با انتخاب سایز و رنگ
- 🛒 **سبد خرید**: مدیریت محصولات در سبد
- 💳 **تسویه حساب**: فرم ثبت سفارش
- 👤 **پروفایل کاربر**: مشاهده و ویرایش اطلاعات + لیست سفارشات
- 🔐 **ورود/ثبت‌نام**: سیستم احراز هویت کامل
- ⚙️ **پنل ادمین**: مدیریت محصولات، سفارشات و کاربران

### قابلیت‌ها
- 🔍 **جستجو و فیلتر**: جستجوی محصولات بر اساس نام، دسته‌بندی، قیمت
- 🏷️ **دسته‌بندی**: مدیریت دسته‌بندی محصولات
- 💰 **قیمت‌گذاری**: قیمت اصلی و قیمت با تخفیف
- 📸 **تصاویر**: آپلود تا 3 تصویر برای هر محصول
- 📦 **موجودی**: مدیریت موجودی انبار
- 🎨 **سایز و رنگ**: انتخاب سایز و رنگ برای محصولات
- 🛒 **سبد خرید**: افزودن، حذف و ویرایش محصولات
- 📋 **سفارش‌ها**: ثبت و پیگیری سفارشات
- 👥 **پروفایل کاربر**: مدیریت اطلاعات شخصی و آدرس

## نصب و راه‌اندازی

### پیش‌نیازها
- Python 3.12+
- Django 6.0.4
- Pillow (برای مدیریت تصاویر)

### مراحل نصب

1. **فعال‌سازی محیط مجازی**:
```bash
cd /home/amirali/Desktop/clothing_store
source /home/amirali/Desktop/futuristic_blog/.venv/bin/activate
```

2. **اجرای مایگریشن‌ها** (در صورت نیاز):
```bash
PYTHONPATH=/usr/lib/python3/dist-packages python manage.py migrate
```

3. **راه‌اندازی سرور**:
```bash
PYTHONPATH=/usr/lib/python3/dist-packages python manage.py runserver 0.0.0.0:8001
```

4. **دسترسی به فروشگاه**:
- آدرس فروشگاه: http://127.0.0.1:8001/
- پنل ادمین: http://127.0.0.1:8001/admin/

## ساختار پروژه

```
clothing_store/
├── clothing_store/          # تنظیمات اصلی پروژه
│   ├── settings.py         # تنظیمات Django
│   ├── urls.py             # URL های اصلی
│   └── wsgi.py
├── shop/                    # اپلیکیشن فروشگاه
│   ├── models.py           # مدل‌ها (Category, Product, Cart, Order, etc.)
│   ├── views.py            # ویوها
│   ├── urls.py             # URL های shop
│   ├── forms.py            # فرم‌ها
│   ├── admin.py            # تنظیمات پنل ادمین
│   └── context_processors.py  # Context processor برای تعداد سبد
├── templates/               # قالب‌های HTML
│   ├── base.html           # قالب پایه
│   ├── shop/               # قالب‌های فروشگاه
│   └── registration/       # قالب‌های احراز هویت
├── static/                  # فایل‌های استاتیک
├── media/                   # فایل‌های آپلود شده
└── db.sqlite3              # پایگاه داده
```

## مدل‌های دیتابیس

### Category (دسته‌بندی)
- نام، slug، توضیحات، تصویر

### Product (محصول)
- دسته‌بندی، نام، توضیحات، قیمت، تخفیف
- تصاویر (تا 3 تصویر)
- موجودی، سایزها، رنگ‌ها
- محصول ویژه

### UserProfile (پروفایل کاربر)
- شماره تلفن، آدرس، شهر، کد پستی
- عکس پروفایل

### Cart (سبد خرید)
- کاربر، تاریخ ایجاد، وضعیت فعال

### CartItem (آیتم سبد)
- سبد، محصول، تعداد، سایز، رنگ

### Order (سفارش)
- کاربر، وضعیت، اطلاعات گیرنده
- آدرس، مبلغ کل

### OrderItem (آیتم سفارش)
- سفارش، محصول، تعداد، قیمت، سایز، رنگ

## ویژگی‌های طراحی

### RTL و فارسی
- کاملاً RTL با Bootstrap 5 RTL
- فونت Vazirmatn برای متن‌های فارسی
- تمام متن‌ها و برچسب‌ها به فارسی

### طراحی مدرن
- رنگ‌بندی گرادیانت زیبا
- انیمیشن‌های hover
- کارت‌های محصول با سایه و افکت
- نوار ناوبری sticky
- آیکون‌های Bootstrap Icons
- طراحی ریسپانسیو کامل

### تجربه کاربری
- نمایش تعداد محصولات در سبد خرید
- پیام‌های موفقیت و خطا
- فیلتر و جستجوی سریع
- نمایش محصولات مرتبط
- نمایش درصد تخفیف
- وضعیت موجودی محصولات
